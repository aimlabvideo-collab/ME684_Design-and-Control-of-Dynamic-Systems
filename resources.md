# Resources

## Software

- **MATLAB / Simulink** — available to CSUN students. See the [CSUN MATLAB portal](https://www.csun.edu/it/software) for installation. The Control System Toolbox is used for `tf`, `ss`, `step`, `rlocus`, `bode`, and `nyquist`.
- **Python** (optional) — `numpy`, `scipy`, `matplotlib`, and the [`python-control`](https://python-control.readthedocs.io/) package reproduce most in-class examples.
